# Astro Chatting

bueno gente aqui les dejo el repositorio oficial de astro chatting.

esto es algo simple pero funcionaal xd

## testear la app
si quieren probarla y ver que onda, la app ya esta subida y corriendo. entren a link oficial aca:
https://www.astrochatting.us/

## algunas de las features que trae
- notas de voz nativas: puedes mandar audios al toque sin salir del chat
- sistema de regalos: podes mandar y recibir items cosmeticos ahi mismo 
- tradeos en vivo: sistema de trading de archivos re avanzado directo en el chat
- soporte completo de multimedia: gifs, stickers integrados, previas de links y demas
- Astro Security: tiene filtros anti spam y protege que no te llenen de basura el canal
- markdown: podes dar formato, meter spoilers, y mencionar usuarios o roles
- puedes crear bots
- puedes configurar avanzadamente tu grupo
- full optimizado en next.js y react para cero lag

Y esas son algunas de las funciones que tienen entre muchaas mas que podran ver en el source

## open source y colaboraciones
lo mas importante de todo es que el codigo es open source y de la comunidad. 

se aceptan modificaciones y se aseguro que se van a poder aplicar. si tenes ganas de sumar una feature nueva, arreglar algo del diseno o tapar un bug, manda PR que aca se revisa y se integra.

como aportar:
1. dale fork al repositorio
2. create una rama para lo tuyo (ej. git checkout -b feature-nueva)
3. hace tus cambios y mandales commit
4. hace push (git push origin feature-nueva)
5. abri el pull request asi lo vemos y lo sumamos a la version oficial

nada mas, estan invitados a meterle mano al codigo y armar el chat del ano entre todos.
